1️⃣ Sao lưu (khuyến nghị làm trước)

Tải toàn bộ dữ liệu (thành viên, điểm danh, thanh toán) ra 1 file JSON để giữ ngoài máy. Mở lại bằng trang xem offline bất cứ lúc nào.

📂 Mở trang xem backup
2️⃣ Lưu trữ & dọn DB (giải phóng dung lượng)

Gộp toàn bộ điểm danh trước hôm nay vào kho lưu trữ (chỉ giữ con số tổng), rồi xóa các dòng cũ khỏi DB để nhẹ.

✓ Báo cáo vẫn hiện TỔNG = live + đã lưu trữ (cộng dồn). Ví dụ: 5 lượt đã lưu trữ + 2 lượt mới = hiện 7.

⚠ Nên bấm "Tải backup JSON" ở trên TRƯỚC khi lưu trữ. Thao tác này xóa dòng điểm danh cũ khỏi DB (không thể hoàn tác).
